#### Drawer Component:(**_WIP_**)

Provides a simple toggle-able drawer component which holds a collection of other React or HTML elements within.

* Empty Drawer:

```jsx
<Drawer title="Section" />
```

* Drawer with some Content

```jsx
<Drawer title="ClickMe">
  <div>
    <p>Hello World!</p>
  </div>
</Drawer>
```
